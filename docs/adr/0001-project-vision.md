# ADR-0001 Project Vision

## Status

Accepted

## Context

Gemoy dibangun sebagai AI Assistant perusahaan yang menjadi pusat pengetahuan (Knowledge Center) dan operasional (Operations Center).

Gemoy bukan chatbot biasa.

Gemoy akan menjadi "Second Brain" perusahaan.

## Decision

Project akan dibangun dengan prinsip:

- Modular Architecture
- Open Source First
- Docker First
- AI Provider Independent
- Data First

## Consequences

Seluruh pengembangan harus mengikuti prinsip di atas.
